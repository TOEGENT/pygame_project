Этот репозиторий содержит студенческий проект, разработанный с использованием Pygame.
Проект создан в учебных целях для практики основ разработки игр на Python.
В рамках работы реализуются базовые игровые механики, обработка событий и отрисовка объектов на экране.
Также уделяется внимание структуре кода, чтобы проект было удобно развивать и поддерживать.
Репозиторий служит площадкой для экспериментов, отработки навыков и демонстрации прогресса в обучении.
Цель проекта — получить практический опыт полного цикла создания небольшой 2D-игры.

## Пример: взаимодействие шариков в 2D по инерции

Добавлен файл `main.py` с простым симулятором:
- шарики движутся по инерции;
- отскакивают от стен;
- сталкиваются друг с другом с упругим обменом импульсом.

### Запуск

1. Установите зависимости:

	pip install pygame

2. Запустите пример:

	python main.py

### Управление

- Закрыть окно: кнопка закрытия окна.
- Сбросить шарики: клавиша `R`.
