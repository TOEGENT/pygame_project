## Описание проекта

Этот проект представляет собой симуляцию взаимодействия шариков в 2D-пространстве с использованием библиотеки Pygame. Основное внимание уделено физике столкновений и механикам, зависящим от массы, скорости и других характеристик объектов.

### Реализованные механики
1. **Физика столкновений**: Учет массы, скорости и упругости при взаимодействии объектов.
2. **Ударный импульс**: Результат столкновения зависит от импульса и порогов цели.
3. **Ассимиляция**: Постепенная передача массы и изменение цвета атакующего.
4. **Дробление**: Разделение объекта на фрагменты при превышении порога разрушения.
5. **Рикошет**: Упругий отскок при недостаточном импульсе.

### Как запустить
1. Убедитесь, что у вас установлен Python 3.10+.
2. Установите зависимости:
   ```bash
   pip install pygame
   ```
3. Запустите основной файл:
   ```bash
   python main.py
   ```

### Управление
- **R**: Перезапуск симуляции с новыми шариками.

### Параметры
- **Количество шариков**: 14 по умолчанию.
- **Минимальный радиус фрагментов**: 9.0.
- **Защита новых фрагментов**: 0.45 секунд.

### Цели проекта
- Изучение основ физики в играх.
- Практика работы с библиотекой Pygame.
- Разработка и тестирование игровых механик.
